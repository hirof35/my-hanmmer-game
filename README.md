Hammer Throw Physics Game (ハンマー投げ物理ゲーム)
Matter.js を使用した、物理演算ベースのハンマー投げゲームです。マウスでハンマーを振り回し、適切なタイミングでリリースして飛距離を競います。

🎮 遊び方
振り回す: ハンマーをクリックしてドラッグし、円を描くようにマウスを動かして回転を加えます。

投げる: 狙った角度でマウスボタンを離すとハンマーが飛び出します。

判定: 飛距離に応じてスコアが計算され、ブラウザのストレージにベストスコアが保存されます。

🛠 開発環境・技術スタック
Engine: Matter.js (物理演算エンジン)

Language: JavaScript (ES6+)

Rendering: HTML5 Canvas

🚀 セットアップ方法
本プロジェクトはフロントエンドのみで動作しますが、ローカル環境で開発・管理するために以下の手順で行ってください。

1. リポジトリの準備
Bash
# プロジェクトフォルダへ移動
cd hammer-game

# npmプロジェクトの初期化
npm init -y

# 物理演算ライブラリのインストール
npm install matter-js
2. 実行方法
プロジェクトのルートに index.html を配置し、ブラウザで開くだけでプレイ可能です。

Bash
# もしNode.js環境でサーバーとして動かしたい場合 (例: serveを使用)
npx serve .
⚙️ ゲームの仕組み
物理挙動: Constraint を使用して、回転の遠心力とリリースの慣性をリアルタイム計算しています。

カメラ制御: ハンマーの飛距離に応じてカメラが滑らかに追従する Lerp (線形補間) 実装を行っています。

データ保存: localStorage を使用し、ブラウザを閉じてもベストスコアが保持される設計です。
